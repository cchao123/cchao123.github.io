# Git相关工具

# 语义版本

版本管理建议遵循 semver 规范

# 辅助工具

使用 gitflow 规范化分支管理流程。

使用 semantic-release 来辅助全自动创建版本号。

使用 commitizen 使代码提交信息规范化。

使用 conventional-changelog 自动生成 changelog 。

使用 husky 便捷添加 Git hooks 。

使用 lint-staged 对待提交文件进行 lint 。
